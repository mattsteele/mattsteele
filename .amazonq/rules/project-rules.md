# Project Rules

## Framework Requirements

- Use [Astro](https://docs.astro.build/en/getting-started/) latest version best practices
- Use [TailwindCSS v4+](https://tailwindcss.com/docs/installation/using-vite) with Vite plugin
- Deploy to [Cloudflare Pages](https://developers.cloudflare.com/pages/) following their guidelines

## Code Standards

- Follow Astro component patterns and file structure
- Use Tailwind utility classes for styling
- Optimize for Cloudflare Pages deployment
- Maintain responsive design principles

## File Organization

- Components in `/src/components/`
- Pages in `/src/pages/`
- Layouts in `/src/layouts/`
- Assets in `/src/assets/`
